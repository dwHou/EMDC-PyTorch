## Requirements  
Required 
* pytorch
* numpy
* pillow 
* opencv-python-headless 
* scipy    
* Matplotlib

## Quickstart

1. **Step 1:** Put the "test_input" folder downloaded and decompressed from the CodaLab under the path "./Submit-test/"
2. **Step 2:**

```shell
cd ./Submit-test
bash testphase.sh
```

3. **Step 3:** Check the results under the path ./Submit-test/results
